# Dashboard (Power BI)

Este repositório inclui a pasta `dashboards/powerbi/` com um guia para construir o PBIX.

Sugestão de páginas:
- Visão geral (KPIs)
- Distribuições (idade, BMI, hábitos)
- Segmentação (sexo, meio de transporte, etc.)
- Correlações (BMI x classes; hábitos x classes)

Dados de apoio:
- Use `data/processed/base_processada.csv` e/ou exports do notebook.
