# Tech Challenge — Obesity (Fase 4)

Este projeto entrega um fluxo **end-to-end** para predição de obesidade a partir da base `Obesity.csv`.

## Objetivo de negócio
Apoiar o time médico e gestores com:
- predição do **nível de obesidade (multiclasse)**
- interpretação de fatores correlacionados (hábitos, histórico familiar, etc.)
- monitoramento via dashboard

## Entregáveis
- Pipeline de dados + treino (scikit-learn / imblearn)
- App Streamlit
- Estrutura para Dashboard Power BI
- Documentação MkDocs
- Roteiro de vídeo

Para rodar localmente: veja o `README.md` do repositório.
