# App (Streamlit)

O app permite inserir as variáveis do paciente e retornar a previsão do **nível de obesidade**.

Rodar localmente:
```bash
streamlit run Predicao.py
```

Para deploy:
- Streamlit Community Cloud: aponte para este repositório
- Garanta que `models/modelo_obesidade.joblib` exista (treine antes ou inclua no repositório, se permitido)
